## Documentation

Les documents présents dans ce dossier ont été réalisés dans le cadre de mon
stage de 3ᵉ année de licence en informatique, au sein d’un SOC.

Ils illustrent :
- la mise en place et l’étude d’une sandbox CAPEv2,
- un exemple d’analyse SOC d’un binaire suspect ayant conduit à la qualification
  d’un faux positif.

Les informations sensibles ont été anonymisées et aucun artefact malveillant
n’est fourni.
